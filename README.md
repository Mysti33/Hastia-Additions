# Features

## Elemental Totems

![Totems](https://media.forgecdn.net/attachments/description/null/description_31656921-c873-4f87-ba3c-3b37ec514c6e.png)

Elemental Totems add a **new survival mechanic** based on powerful artifacts.

Each totem grants **unique abilities** that can help you survive dangerous situations.

### Fire Totem

The Fire Totem allows you to **throw fireballs**.

Additional effects:

* Fire immunity
* Temporary health boost when taking damage

---

### Earth Totem

The **most powerful totem**.

Abilities:

* Create an explosion
* If you die while holding it, you receive an **upgraded Totem of Undying effect**

Very useful in **PvP situations**.

---

### Air Totem

The Air Totem grants **Feather Falling** when used.

This allows you to:

* Escape during combat
* Perform large jumps safely

---

# Spells

You can unlock powerful spells and **select them using the T and R keys**.

### Fire Spell

Launch a magma block that damages enemies or explodes on impact.

⚠️ Does **not destroy blocks**.

Unlock condition:

* Discover the Nether.

---

### Repel Spell

Pushes all nearby entities away within a **10 block radius**.

Very useful in dangerous combat situations.

Unlock condition:

* Discover a **Nether Fortress**.

---

# Special Effects

Special Effects are **semi-permanent upgrades** obtained by spending resources.

Open the effects menu with:

O

Examples of available effects:

* Regeneration → Faster healing
* Resistance → Passive damage reduction
* Dragon Effect → Allows you to throw dragon fireballs

⚠️ Effects are **lost when you die**.

More effects will be added in future updates.

# Reinforced Tools

![Reinforced Tools](https://media.forgecdn.net/attachments/description/1362880/description_b329917f-e9df-4dfd-b93a-854219775fa9.png)

The mod introduces **reinforced tools**, stronger than Netherite.

### Reinforced Pickaxe

* Extremely fast mining
* Efficiency V speed
* Very high durability

---

### Reinforced Shovel

* Faster digging
* Instant block breaking for many blocks

---

# Invisible Light Blocks

This mod allows players to **craft invisible light blocks**.

Normally these blocks are only available using commands.

This is especially useful for players who want **clean lighting in survival builds**.

# Installation

Requirements:

* Minecraft 1.20.1
* Minecraft Forge

Steps:

1. Install Forge.
2. Download the mod JAR file.
3. Place it inside the `mods` folder.
4. Launch Minecraft.
